# TwanBase
Android基础项目,已集成常用组件,便于快速开发.

## 1. 已集成:
- 1.canary
- 2.rxandroid
- 3.butterknife
- 4.retrofit (网络)
- 5.SmartRefreshLayout (刷新)
- 6.gson
- 7.rxpermissions2 (权限)
- 8.bottom-bar (底部栏，80%的app都需要)
- 9.swipebacklayout (左滑手势)
- 10.autodispose (处理rxjava内存泄漏)

<br>**本库只集成app必备的库，里面也集成了少量的工具类，其他的好库陆续添加在本文档!**
